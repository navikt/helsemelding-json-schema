# helsemelding-json-schema

Repository for schema generation, validation and distribution of Helsemelding JSON Schemas.

## Modules

### json-schema-core

Responsible for:

* Schema source models
* JSON Schema generation
* Schema export
* JSON validation against published schemas

See: `json-schema-core/README.md`

### json-schema-server

Responsible for:

* Serving published schemas over HTTP
* Schema discovery
* Version lookup

See: `json-schema-server/README.md`
